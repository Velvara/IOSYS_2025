Velvara Oceanside Town Modular Building Kit
-------------------------------------------

Thank you for purchasing the Oceanside Town Kit! This pack is made for a PBR workflow, but feel free to experiment in your own styles!

A couple of things to note:

1. The Master shader uses a Overlay Mask texture that you may change or adjust to create dirt, leaks, wear or other layers to the material.

2. The maps provided for smoothness were created as roughness maps. So the shader inverts a roughness map to make it into a smoothness map.

3. If using URP, unpack "Materials_URP.zip" and replace the "Materials" folder, both in "Velvara > Oceanside Town". Demo scene will need to be set up with new lights.

*Asset preview scene included, Demo scene not included.
